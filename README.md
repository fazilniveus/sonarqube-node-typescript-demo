# sonarqube-node-tpescript-demo

Demo to setup sonarqube for Node Typescript project.

## Run the application

```
$ npm install
$ curl -sSL https://raw.githubusercontent.com/bitnami/bitnami-docker-sonarqube/master/docker-compose.yml > docker-compose.yml
$ docker-compose up -d
$ npm run sonar
```
